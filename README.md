# NEBULA-X 🌌

**Orbital Antimatter Energy Station for Clean Power and Interstellar Propulsion**

## 🚀 Tech Stack
- Next.js + React
- Tailwind CSS
- Vercel for hosting

## 🛠️ Getting Started
```bash
yarn install
yarn dev
```

## 🔗 Deployment
Automatically deployed to [https://nebula-x.vercel.app](https://nebula-x.vercel.app)

## 📂 Structure
- `/components`: Modular UI
- `/pages`: Routing
